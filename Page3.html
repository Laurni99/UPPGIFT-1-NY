Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Id och Class är två attributer med olika funktioner.En id används när det bara finns en unik element som tillexempel titeln på sida 3 "CSS" och en class kan användas när det finns flera element med liknande funktioner, de börjar även på olika sätt. En id börjar med en fyrtagg och en class börjar med punkt. en id används när något specifikt på webbplatsen ska identifieras och class kan användas en eller flera gånger beroende på om en önskar att ha samma stil på elementen

Vad är en regel i CSS och vad består den av?

Css har regel som ger instruktioner på hur ett element ska se ut på en webbsida. Den består av elektorer och deklarationsblock
